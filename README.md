# AI Resume Analyzer + ATS Score

## Description
This project analyzes resumes and compares them with job descriptions to generate an ATS score and improvement suggestions using NLP.


## Features
- Upload resume (PDF)  
- ATS score calculation  
- Missing keyword detection  
- Improvement suggestions  


## Tech Stack
- Python  
- Streamlit  
- Scikit-learn (TF-IDF)  
- PyPDF2  


## Project Structure
-app.py
-requirements.txt
-README.md

## Installation
pip install -r requirements.txt


## Run the Project
streamlit run app.py


## How It Works
- Extracts text from resume  
- Compares with job description  
- Calculates similarity score  
- Displays missing keywords
